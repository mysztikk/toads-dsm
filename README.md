# toads-dsm
Battletoads disassembly project. All tools made by me, clean room, (no stack
overflow, no ai, no nonsense) and in x86-64 assembly.

## Project Goals
- Fully disassemble and document on a micro and macro scale Battletoads for the
NES.
- Built every tool for the process entirely on my own in x86-64 assembly (except
an emulator)
